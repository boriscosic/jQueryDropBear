DropBear jQuery Plugin

Version 1.0

This plugin allows inline embedding of menu and toolbars. I needed something that was minimal and easy to configure to display inline menus for links. While I was working on this I also ran into an issue with "select" tags which I needed to extend into drop downs. This is a another great drop down plugin http://code.google.com/p/dropdown-check-list/ but I didn't like it's reliance on jQuery UI and it wasn't quite there for reusing a single plugin for all my drop downs.

How It Works

This plugin looks for a child div for all all a elements by id referenced through href tag. It doesn't have to be an a element since you can call the plugin on anything you want (not very well tested). When replacing the select tags, the plugin will generate replacement DOM and hide the element. If your select list has multiple attached to it, selection items are automatically converted to checkboxes and by default to radio input.

How To Use It

This is the minimal invocation of DropBear. 
		$(function() { 
			$('a.dropbear').dropbear();
		});
Initiate DropBear with callback. 
		$(function() { 
			$('a.dropbear').dropbear({fnHide:dropBear});
		});
		
		function dropBear(values) {
			alert('Selected: '+values);
		}
Use alternative invocation such as mouseover. 
		$(function() { 
			$('a.dropbear').dropbear({showTrigger:'mouseover'});
		});

Options

  • showTrigger: default 'click'. Change the trigger for drop down invocation.
  • hideTrigger: default 'mouseout'. Change the trigger for removing the drop down.
  • fnHide: default null. Set callback when drop down hides.
  • position: default 'absolute'. If you need to change the positioning of the drop down. This can also be done in CSS.
  • linkTitle: default 'Select'. If you want to override the default title. Useful for select elements.
